# Jasnoor IP Locator (Bash)

A lightweight Bash-based IP geolocation CLI tool that retrieves detailed network and location information using a public API.

## Features
- Continent, Country, Region, City, ZIP
- Latitude & Longitude (approximate)
- Timezone & Currency
- ISP, Organization, ASN details
- Mobile / Proxy / Hosting detection
- No external dependencies

## Usage

```bash
chmod +x get_locator.sh
./get_locator.sh 8.8.8.8
